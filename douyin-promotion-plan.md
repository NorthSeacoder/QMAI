# 青幕AI写作 —— 抖音推广方案

## 一、账号定位

**账号名称建议**：青幕AI写作 / AI写小说的莫初 / 网文AI工具人

**人设定位**：一个不会写代码的普通人，用AI做出了一款AI写小说软件，现在用它辅助写长篇网文。

**核心标签**：
- AI写作工具
- 网文作者必备
- Vibe Coding
- AI辅助创作
- 长篇小说

**差异化卖点**（区别于市面上其他AI写作工具）：
> "别的AI写作工具写到第50章就忘了第3章的事，我的不会。"

## 四、视频制作要点

### 拍摄/录屏要点

1. **录屏为主**：软件操作画面 + 画外音解说（或文字旁白）
2. **节奏要快**：每3-5秒一个信息点，不能有空白等待
3. **关键操作放大**：用动画/高亮标注当前在做什么
4. **对比展示**：有问题的效果 vs 解决后的效果

### 前3秒钩子模板

```
❌ "大家好，今天给大家介绍一款AI写作软件……"
✅ "AI写了100章小说，到第101章——它忘了主角叫什么。"
✅ "我花4个月做了个东西，它能让AI记住200万字。"
✅ "所有AI写小说的人都会遇到这个问题。"
✅ "角色A不知道这个秘密——但AI让他说出来了。"
```

### 文案结构（30-60秒视频）

```
[0-3s]  钩子（抛出问题/反差）
[3-8s]  痛点放大（引发共鸣）
[8-25s] 解决方案展示（软件操作录屏）
[25-28s] 效果对比（前后对照）
[28-30s] CTA（引导评论/关注/主页链接）
```

### 常用CTA话术

- "链接在主页，免费开源的"
- "评论区扣1，我发你下载地址"
- "关注我，下期教你怎么配置"
- "你用AI写小说遇到过什么问题？评论区聊聊"

## 八、数据观察与迭代

### 关注指标

| 指标 | 目标 | 说明 |
|------|------|------|
| 完播率 | >30% | 说明内容节奏没问题 |
| 点赞率 | >3% | 说明内容引发共鸣 |
| 评论率 | >1% | 说明内容有讨论度 |
| 主页访问 | 每条视频>50 | 说明有人想进一步了解 |
| GitHub Star增长 | 每周观察 | 最终转化指标 |

### 迭代策略

- 前10条视频重点测试：钩子型(A) vs 功能演示(B) vs 干货(C)，看哪类数据好
- 数据好的类型加大产出
- 数据差的分析原因：是钩子不行？还是内容本身不吸引？
- 每周复盘一次，调整下周内容计划

## 九、低成本起步方案

如果精力有限，最小启动方案：

1. **第一周只发3条**：选A1、A2、B2这三个方向
2. **录屏工具**：OBS免费录屏 + 剪映免费剪辑
3. **不出镜**：纯录屏+文字旁白或画外音
4. **不花钱投流**：先靠自然流量测试内容方向
5. **时间投入**：每条视频1-2小时（录屏30分钟+剪辑1小时）

### 视频制作最简流程

```
1. 写好30秒口播文案/旁白文字
2. OBS录屏操作演示（可以2倍速加快无聊部分）
3. 剪映剪辑：加字幕+加背景音乐+掐掉空白
4. 导出发布
```

## 十、风险与注意事项

1. **不要过度承诺**："辅助写作"而非"自动写书"，避免用户期望过高导致差评
2. **不要贬低其他工具**：只讲自己的优势，不直接对比攻击竞品
3. **注意AI相关政策**：抖音对AI生成内容有标注要求，展示时说明是"辅助工具"
4. **代码演示要脱敏**：展示软件时注意不要暴露API Key等敏感信息
5. **评论区控场**：遇到杠精不要硬刚，专注回复真实用户的问题
6. **版本稳定性**：推广前确保当前版本没有明显bug，第一印象很重要
